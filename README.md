# Monster Chase

A 2D arcade-style game developed as a Computer Graphics assignment for Semester 5.

## Description

Monster Chase is a Unity-based game where players navigate through levels while avoiding enemies. The game features animated sprites, multiple enemy types, and engaging gameplay mechanics.

## Technologies Used

- **Unity Game Engine**
- **C# Programming**
- **2D Graphics & Animation**

## Project Structure

- `Assets/Scripts/` - Game logic and mechanics
- `Assets/Sprites/` - 2D artwork and graphics
- `Assets/Animations/` - Player and enemy animations
- `Assets/Scenes/` - Game levels
- `Assets/Prefabs/` - Reusable game objects
